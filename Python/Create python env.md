## **Virtual env setup:**
- venv
	- `python3 -m venv .venv`
	- `source .venv/bin/activate`
- env 
    - `pip install virtualenv`
    - `virtualenv env`
    - `source env/bin/activate`

## **Jupyter setup:**
    - `pip install notebook`
    - `jupyter notebook`


### Python verions
`brew install pyenv`
`pyenv install 3.9`
`pyenv global 3.9 `
